# Zecur (Zero-Cursor)

Zecur adalah *AI-powered Agentic IDE Extension* untuk VS Code. Proyek ini bertujuan untuk membawa kemampuan "Composer" (seperti di Cursor) menjadi *open-source* dan bisa dikembangkan oleh siapa saja.

## Visi
Membangun asisten koding yang bisa berpikir, menulis kode di banyak file sekaligus, dan mengeksekusi perintah terminal secara mandiri.

## Struktur Utama
- `src/agent/`: Otak AI (Recursive Loop).
- `src/tools/`: Tangan AI (File System & Terminal Access).
- `src/webview/`: Mata AI (UI/Chat Panel).

## Cara Pengembangan
1. Jalankan `npm install`.
2. Buka di VS Code.
3. Tekan `F5` untuk menjalankan Extension Development Host.
